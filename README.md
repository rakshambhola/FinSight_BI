# FinSight_BI
FinSight BI short for *Financial Insight using Power BI* is an automated financial analysis project that fetches company financial data from online sources, processes it using Python, and visualizes key metrics through Power BI dashboards. The project focuses on reducing manual effort in financial analysis by enabling automated data updates and structured visuals of company performance.
